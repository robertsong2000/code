# 网页链接提取器

一个简单的网页工具，用于提取指定网页中的所有链接。

## 项目介绍

这个项目是一个基于浏览器的工具，可以帮助用户从任何网页中提取所有超链接。它使用 CORS 代理来解决跨域访问限制，提供了简单直观的用户界面。

## 功能特点

- 支持输入任意网页 URL
- 自动提取页面中的所有超链接
- 使用 CORS 代理解决跨域问题
- 实时显示提取结果
- 支持在新标签页中打开提取到的链接

## 安装说明

1. 克隆项目到本地：
